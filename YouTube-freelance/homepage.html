General instructions - gulp starter pack

PNG icons

For png icons use i.icon.icon-name . Png icons should be regular icon and @2x postfix

Folder: /src/icons

SVG icons

For svg icons use pug mixin +svgMono(nameOfIcon) or +svgColor(nameOfIcon). You can customize it as it would be regular font. font-size property should be set as icon height

Please keep in mind exoprting svg. Sketch is good in all cases. Photoshop + illustrator - use quick export function (CC2017+) with ctrl+c ctrl+v of layers You will need viewBox property and icon should be bounded to artboard. Otherwise it will be as small as artboard differance relative to icon size

Folder: /src/icons
